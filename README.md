# Face Verification using MobileNetV2 and LFW Dataset

Bu proje, LFW veri seti üzerinde MobileNetV2 mimarisi kullanılarak geliştirilmiş bir yüz doğrulama sistemidir.

## 🚀 Özellikler
- **Model:** MobileNetV2 (Feature Extractor)
- **Doğrulama Metriği:** L2 (Euclidean) Distance
- **Veri Seti:** LFW (Labeled Faces in the Wild)
